# Gestión de Libros EduLend

Este proyecto es una aplicación de consola desarrollada en **Python** que permite gestionar un inventario básico de libros.
El sistema permite registrar libros, actualizar cantidades, buscar libros y gestionar préstamos.

## Funcionalidades

El sistema incluye un menú interactivo con las siguientes opciones:

1. **Ingresar libro**
   Permite registrar un nuevo libro en el inventario con su cantidad y categoría.

2. **Actualizar cantidades**
   Permite modificar la cantidad disponible de un libro existente.

3. **Mostrar inventario**
   Muestra todos los libros registrados en el sistema.

4. **Buscar libro por nombre**
   Permite consultar la información de un libro específico.

5. **Prestar libro**
   Reduce la cantidad disponible de un libro cuando es prestado.

6. **Salir**
   Finaliza la ejecución del programa.

# 🛠 Requisitos

Antes de ejecutar el proyecto asegúrate de tener instalado:

* **Python 3.10 o superior**

Puedes verificarlo con:

```bash
python --version
```

o

```bash
python3 --version
```

---

# Clonar el repositorio

Clona el repositorio utilizando **git**:

```bash
git clone https://github.com/Jcmedinah/Momento1_nuevas_tecnologias
```

Luego entra a la carpeta del proyecto:

```bash
cd edulend-libros
```

---

# Ejecutar el proyecto

Para ejecutar la aplicación usa:

```bash
python main.py
```

o

```bash
python3 main.py
```

Esto iniciará el **menú interactivo de gestión de libros**.
